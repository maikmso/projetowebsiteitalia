# **5 CIDADES MARAVILHOAS PARA COLOCAR NO SEU DESTINO DE VIAGEM!!!**

*Projeto de um website contendo 5 cidades da Itália como recomendação para futuros viajantes, mostrando bastante conteúdo sobre cada uma dessas cidades. Aqui eu coloquei algumas das comidas típicas, lugares turísticos e hospedagens de cada uma das cidades, para a pessoa procurar com os melhores preços em cada site de hospedagem.*
